# Coaching Library

This directory holds teaching content extracted from Rainbow Six Siege coaching video sessions. It is not currently displayed on the site but is preserved here for a future coaching section.

## Structure

Each file corresponds to a video session (e.g. `V1.md`, `V2.md`). Tips are tagged with IDs like `V2.T01` that were previously referenced in player teaching priority lists.

## Adding files

Copy the relevant session markdown files from the Knowledge Base into this directory. The site will ignore them until a coaching section is built.
